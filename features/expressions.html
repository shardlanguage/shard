Values

In an expression, you can use the following values:

  • Decimal number (1234)

  • Float number (12.34)

  • Binary number (0b10)

  • Octal number (0o1234)

  • Hexadecimal number (0x1234ABCD)

  • Characters ('c')

  • String (limited) ("str")

  • Identifier (x)

  • Function call (x())

  • Reference (&x)

  • Array value (x[y])

  • Expressions (a+b, a/b*(c-d), …)
